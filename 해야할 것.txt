★ 고도화(추가기능)

1. GPS 인식하여 병원 조회시 해당 위치의 주소를 조건 칸에 자동 입력
> GPS API 기능 찾아 적용
> user_info / hospital_info(GPS 허용여부 컬럼 추가)
> 회원 가입시 GPS 현재 위치 인식 허용 동의 추가 

2. 음성인식 
> 시니어 고령 사용자 병원 예약시 음성을 인식하여 진료 예약 처리

3. 아이디 찾기/비밀번호 찾기 
> 라벨만 만들어 두고 추후 기능 적용
 
4. 기관ID 찾는 PROCESS 
> 각 병원마다 공공데이터에 PKEY인 기관ID를 알고 있는 지 의문이라.. 모르면 추가 예상
