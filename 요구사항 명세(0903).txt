공통 - 로그인 / 회원가입 / 마이페이지


사용자
- 병원 목록 조회
- 예약(알림톡 수신 여부 체크) / 변경(예약시간 1시간 전에는 변경 불가) / 취소 
- 진료일 도래시 알림톡 수신(하루 전, 1시간 전)
- 예약 내역 조회

관리자 
- 예약 발생시 알림톡 수신 
- 예약 목록 조회
- 예약 변경 

1. 메인 페이지
회원가입 -> 로그인 -> 메인페이지 
* 회원가입시 gps정보 이용 동의 여부 
* 사용자/ 관리자 계정에 따라 상단 메뉴 별도 구성

2. 예약하기 메뉴 
시, 도 선택 / 시, 군, 구 선택 / 진료 날짜 선택 / 진료 받을 시간 선택 
진료 날짜 :  캘린더, 확정된 날짜는 block 처리
진료 받을 시간 : 30분 단위 24시간 표시

* 검색 버튼 클릭시 해당 조건에 맞는 병원들만 하단부 출력

★ 보완 : 현장 접수 인원 포함 대기시간에 따라 색상으로 표시 (혼잡,  보통, 여유)
-> 관리자가 지금 바쁘다. 그럼 혼잡 설정하면, 빨간색으로 표시해줌.
★ 현위치 클릭시 gps 정보 받아서 지역정보 검색 조건에 자동 입력 기능
 
개당 병원 클릭시 팝업창 
검색 조건 입력한 예약 정보 가져와서 보여주기, 알림톡 수신여부,
예약시간 1시간 전 변경 불가 안내, 예약하기 버튼
-> 예약 버튼 클릭시 예약 완료 안내 메세지, 예약은 자동 확정처리 안내
